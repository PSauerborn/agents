# PROC-001 - Reduced Flow

## Flow Overview

**Overview**: Reduced flow used to handle small tasks. Single work plan, deconstructed into a single task.

**Scale**: Small tasks (1 file modified)

**Required Subagents**: `work-planner`, `task-decomposer`, `task-executor`

## Workflow

| Step | Agent | Purpose | Output |
| ------ | ------- | --------- | -------- |
| 1 | work-planner | Analyses requirements from `requirements-analyzer` and generates a work plan plan. | Work plan, written to the `documentation-criteria` canonical work-plan location. |
| 2 | (orchestrator) | Wait for user approval of work plan generated by `work-planner`. **[STOP]** | |
| 3 | task-decomposer | Decompose work plan into a single task for execution. | A single executable task file, written to the `documentation-criteria` canonical task location. |
| 4 | task-executor | Execute task file. | |
